## English

### Where to Get Help

- For bugs and compatibility problems: use [GitHub Issues](https://github.com/staklab/PrismRail/issues)
- For feature requests or site support ideas: use [GitHub Issues](https://github.com/staklab/PrismRail/issues/new/choose)
- For contribution guidance: read [CONTRIBUTING.md](./CONTRIBUTING.md)

### Before Opening an Issue

- Confirm you are using the latest code from `main`
- Check whether the issue already exists
- Include browser, site, and reproduction details

### 如何获得帮助

- Bug 和兼容性问题：使用 [GitHub Issues](https://github.com/staklab/PrismRail/issues)
