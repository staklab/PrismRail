# Roadmap

**Language / 语言**

[English](#en) | [中文](#zh)

<a id="en"></a>
## English

### Current Focus

- Improve adapter resilience for supported LLM sites
- Continue refining the glass timeline interaction and spacing
- Strengthen packaging and release-readiness for browser stores
- Expand regression coverage for timeline sampling and jump behavior

### Next Milestones

- More robust fallback selectors for DOM changes
- Better visual tuning across desktop, tablet, and narrow mobile layouts
- Store listing polish and submission assets refinement
- Optional settings improvements for advanced users

### Principles

- Keep the UI subtle and low-intrusion
- Preserve the local-first privacy model
- Prefer maintainable adaptation over fragile site-specific hacks

## 中文

### 当前重点

- 提高已支持 LLM 站点的适配稳定性
- 持续打磨玻璃感时间轴的交互和间距细节
- 完善浏览器商店所需的打包与发布准备
- 扩充时间轴采样与跳转行为的回归覆盖

### 下一阶段里程碑

- 针对 DOM 变动提供更稳健的降级选择器
- 继续优化桌面端、平板和窄屏下的视觉表现
- 完善商店文案、素材和提交流程
- 为高级用户补充更多可配置项

### 设计原则

- 保持低侵入、低打扰的界面风格
- 坚持本地优先与隐私优先
- 优先选择可维护的适配方案，而不是脆弱的站点特判
