# QA Checklist

## 功能正确性

- 能在 ChatGPT、Gemini、Claude 页面识别出消息列表
- 右侧时间轴出现位置不与原生滚动条重合
- 点击节点可以跳转到正确消息
- 当前可见消息对应节点会高亮
- 悬浮预览显示本地摘录文本

## 性能风险

- 内容脚本不会高频全量重建面板
- 滚动监听已节流或经 `requestAnimationFrame` 合并
- 大量代码块场景下折叠逻辑不阻塞主线程
- 未出现明显布局抖动

## 跨站点兼容

- ChatGPT 基础线程可用
- Gemini 基础线程可用
- Claude 基础线程可用
- 某站点无法识别时不会破坏原页面

## 跨浏览器兼容

- Chrome 解包加载成功
- Edge 解包加载成功
- Safari 转换流程具备明确文档

## 设备适配

- 桌面端时间轴不会遮挡输入区或站点悬浮按钮
- 手机端节点仍可点击
- iPad 竖屏和横屏均不影响主内容浏览

## 开源风险

- README 说明了能力边界
- 项目包含免责声明
- 项目未宣称与目标站点官方合作
- 未默认采集、上传或出售用户会话数据

## 免责声明检查点

- 明确本项目为第三方开源工具
- 明确不保证对所有站点版本永久兼容
- 明确用户需自行评估隐私与合规要求
