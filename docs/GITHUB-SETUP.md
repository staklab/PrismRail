## First Release

Suggested release title:

`PrismRail v0.1.0`

Suggested release description:

```markdown
## PrismRail v0.1.0

Initial public release of PrismRail.

### Highlights

- Glass-style right-side timeline for long AI conversations
- Fixed 12-marker navigation model
- Start/progress markers jump to segment starts
- Final marker jumps to the end of the latest reply
- Local hover previews with no paid API
- Support for ChatGPT, Gemini, Claude, DeepSeek, Doubao, GLM, and Grok

### Included

- Source code
- Documentation
- Contribution guide
- Issue and pull request templates

### Notes

This release currently targets Chromium-based browsers.
Safari packaging remains a separate step.
```
