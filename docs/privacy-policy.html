PrismRail Privacy Policy

Last updated: 2026-03-30

Summary

PrismRail is a local-first browser extension for navigating long AI conversations. Its core features work directly in the page and do not require an account, backend service, or external AI API.

What data PrismRail accesses

PrismRail may access visible content on supported AI conversation webpages in order to:

  • identify conversation blocks
  • build the fixed 12-point timeline
  • generate local preview text
  • improve scrolling and rendering behavior on long threads

PrismRail also uses browser extension storage to save local settings such as:

  • whether the extension is enabled
  • preview length
  • code block collapse preferences
  • heavy-thread threshold values

What PrismRail does not do

  • require account registration
  • send conversation content to PrismRail servers
  • use a remote AI API for summaries or navigation
  • sell user data
  • use user data for advertising
  • share conversation content with third parties as part of normal operation

Data storage

Settings are stored locally through browser extension storage APIs. Conversation content is processed in-page for feature execution and is not intentionally persisted to a PrismRail-controlled remote service.

Permissions

PrismRail requests host permissions only for supported AI sites so it can inject the extension on those pages and provide its user-facing functionality.

Third-party services

The core release of PrismRail does not rely on third-party analytics, ad tech, or remote execution services.

User controls

  • disable the extension
  • remove the extension from the browser
  • clear extension storage through browser tools
