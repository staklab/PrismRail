# PRD

## 1. 项目概述

- 项目名称：PrismRail
- 项目定位：面向 ChatGPT、Gemini、Claude、DeepSeek、豆包、GLM、Grok 等 LLM 在线对话网页的开源浏览器插件
- 核心目标：在长对话场景下缓解滚动卡顿，并提供可点击跳转的右侧时间轴导航

## 2. 背景与用户痛点

- 长对话会持续累积富文本、代码块、表格、图片与数学公式，浏览器渲染压力快速上升。
- 原页面默认依赖连续滚动定位历史消息，查找上下文效率低。
- 页面越长越容易出现明显掉帧，尤其在低配电脑、移动端与 iPad 上体验更差。
- 用户需要一种低成本、无 API 依赖、可开箱即用的方案。

## 3. 产品目标

- 在不侵入目标网站核心业务逻辑的前提下，降低长对话页面的渲染负担。
- 提供独立于原生滚动条的右侧时间轴导航。
- 用本地文本摘录实现节点预览，避免额外 API 成本。
- 支持 Chrome、Edge、Safari，兼容 PC、手机端、iPad 端浏览体验。
- 说明：响应式体验覆盖 PC、手机端、iPad 端，但移动设备上的“可安装性”仍受各浏览器平台政策限制。

## 4. 范围

### In Scope

- 长对话页面消息识别
- 渲染性能优化
- 右侧时间轴导航
- 节点悬浮预览的本地免费方案
- 跨站点适配层
- 开源免责声明与使用规范

### Out of Scope

- 云端账号系统
- 付费 API 摘要服务
- 改写目标站点后端接口
- 保证所有未来 DOM 改版下 100% 无维护成本

## 5. 核心功能

### 5.1 长对话性能优化

- 为消息节点增加安全的离屏渲染优化策略
- 对超长代码块进行可折叠处理
- 对图片等媒体元素开启惰性加载提示
- 当消息量较大时自动进入重负载模式

### 5.2 右侧时间轴

- 固定在页面右侧并左移，避免与原生滚动条重叠
- 固定渲染 12 个节点，依次为首对话、10% 到 100% 的分段起点、最后回复尾部
- 首对话与百分比节点跳转到对应内容段落的开头
- 尾部节点跳转到最后一段回复的尾部
- 节点支持高亮当前阅读位置
- 节点悬浮展示本地截取的摘要片段

### 5.3 跨站点适配

- 通过 Adapter 层适配 ChatGPT、Gemini、Claude、DeepSeek、豆包、GLM、Grok
- 允许后续通过新增适配器支持更多 LLM 网站

### 5.4 开源配套

- 提供免责声明
- 提供使用规范
- 提供测试与验收清单

## 6. 角色分工与阶段交付

### Agent1：需求与进度治理

- 负责 PRD
- 负责里程碑与验收定义
- 监测模块化开发是否偏离目标

### Agent2：技术架构

- 负责模块边界、扩展架构、适配策略与性能方案
- 与 Agent1 就可行性和范围做收敛

### Agent3：UI 设计

- 负责时间轴视觉与交互规范
- 负责响应式与可访问性设计

### Agent4：技术实现

- 负责插件脚手架与核心代码

### Agent5：审查与测试

- 负责 review、测试清单、风险检查

## 7. Agent1 与 Agent2 共识

- 第一阶段采用“安全优化优先”策略，避免直接破坏宿主网页框架内部状态。
- 以 `content-visibility`、contain、代码块折叠、本地摘要时间轴作为 MVP 主轴。
- 更激进的 DOM 冻结或消息归档能力保留为后续实验能力，不作为首版默认行为。

## 8. 模块划分

- 站点适配层
- 消息扫描与数据抽取层
- 性能优化层
- 时间轴 UI 层
- 配置存储层
- 后台与设置页
- QA 与文档层

## 9. 里程碑

1. 完成 PRD、架构和 UI 规格
2. 搭建插件骨架和构建脚本
3. 完成 ChatGPT/Gemini/Claude 的基础适配
4. 完成性能优化 MVP
5. 完成时间轴导航与本地预览
6. 完成测试、免责声明、开源说明

## 10. 主要风险

- 目标站点 DOM 结构可能频繁变化
- Safari 与移动端浏览器扩展能力存在差异
- 过于激进的 DOM 改写可能破坏宿主站点交互
- 单纯插件侧优化无法完全替代宿主站点的原生虚拟列表实现

## 11. 验收标准

- 在目标站点长对话页面成功识别主要消息节点
- 页面右侧出现独立时间轴，且不与原生滚动条重合
- 时间轴固定为 12 个节点，语义与位置稳定
- 点击首对话和百分比节点时可稳定跳转到段首，点击尾部节点时可稳定跳转到最后回复尾部
- 悬浮预览无需外部 API 即可显示文本摘录
- 插件在 Chrome/Edge 可作为解包扩展加载
- 项目包含架构、UI、QA、免责声明等开源文档
